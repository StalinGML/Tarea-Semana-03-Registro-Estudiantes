## Descripción del programa
Programa en C# que registra la información de un estudiante aplicando estructuras de datos.  
Utiliza un array para almacenar tres números de teléfono y una clase Estudiante que encapsula todos los datos personales.  
El programa principal solicita la información por consola y luego muestra los datos registrados.

## Funcionalidades
**Clase Estudiante:**
- Atributos: ID, nombres, apellidos, dirección.
- Arreglo string[] para almacenar tres teléfonos.
- Método MostrarDatos() para imprimir toda la información. 

**Programa principal:**
- Captura datos mediante consola.
- Llena el array de teléfonos con un ciclo for.
- Crea un objeto Estudiante.
- Muestra los resultados registrados.
